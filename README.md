# Whac-A-Hole 🐹🔨

一个有趣的 VS Code 插件！在状态栏右下角显示一只地鼠，每次你在编辑器中敲击键盘，地鼠就会被锤子敲到！

## 功能

- 🐹 状态栏右下角显示一只可爱的地鼠图标
- 🔨 每次敲击键盘，地鼠图标会变成被锤子敲击的状态（300ms 后恢复）
- 📊 点击地鼠图标可以查看你总共锤了多少次地鼠
- ⌨️ 监听文本编辑和键盘选择变化事件

## 使用方法

1. 安装插件后，地鼠图标会自动出现在状态栏右下角
2. 在编辑器中输入任何文字，地鼠就会被锤！
3. 点击地鼠图标查看锤击统计

## 开发

```bash
# 编译
npm run compile

# 监听模式
npm run watch
```

按 F5 在 VS Code 中启动调试模式来测试插件。
